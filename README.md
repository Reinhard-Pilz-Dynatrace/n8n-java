# E-Commerce Workshop Demo

A Java-based demonstration application for Dynatrace monitoring workshops.

## Purpose

This repository contains a workshop application that demonstrates performance monitoring and automated remediation workflows. The application is designed to work with Dynatrace observability and N8N automation.

## Quick Start

**Prerequisites:** Java 17 or higher

**Run:**
```bash
# Windows
mvnw.cmd clean compile exec:java

# macOS/Linux
./mvnw clean compile exec:java
```

The application starts an HTTP server on port 8080.
